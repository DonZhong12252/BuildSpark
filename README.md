# BuildSpark

Find your next Codex project.

BuildSpark is a static website for discovering Codex project ideas, copying ready-to-use prompts, and saving a personal build library.

Live site: https://donzhong12252.github.io/BuildSpark/

GitHub repo: https://github.com/DonZhong12252/BuildSpark

## Features

- Curated project ideas for apps, dashboards, AI tools, business tools, games, and beginner websites
- Search, category, difficulty, collection, and build-mode filters
- Daily build challenge
- Project matchmaker
- Saved favorites
- Custom idea submission
- Quick, detailed, and production-ready prompt levels
- One-click starter briefs
- Local browser storage with no account required

## Open Locally

```bash
python3 -m http.server 4177 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4177/
```

## Publish

This is a dependency-free static site. You can upload `index.html`, `styles.css`, and `app.js` to any static host, including GitHub Pages, Netlify, Vercel, Cloudflare Pages, or a normal web server.

User-created ideas and favorites are stored in the visitor's browser with `localStorage`.
